## Project Introduction

### 얼굴인식/검출 기반 AI 퍼스널 컨설팅 어플리케이션

 기존의 얼굴 인식 서비스들과 달리 정면 뿐 아니라 측면까지 인식하여 랜드마크를 검출 후 분석하여 개인별 얼굴의 특징을 찾아주고 그에 맞는 어울리는 스타일 컨설팅을 제공한다.
